# Programmaboekjes

Plaats hier de scans van AZ programmaboekjes.

De bestandsnamen moeten exact overeenkomen met de waarden in `ProgrammaBestanden`
in `src/data/wedstrijden.json`.

Voorbeeld:
- `11-apr-1955 Blauw-Wit uit.webp`
- `15-jan-1961 DWS-uit.webp`
- `30-jan-2000 Feyenoord uit beker.webp`

Aanbevolen formaten: WebP voor compressie, JPEG voor compatibiliteit.
Aanbevolen resolutie: 1200px brede long-edge voor cover-weergave.
